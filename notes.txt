cors -> cross origin resource sharing 

if client and server aren't on same domain (localhost:3000) 
CORS error will appear

Create a middlerware for handling cross-origin errors 
app.use( (req, res, next) =>{
    res.setHeader('Access-Control-Allow-Origin', '*');
    // or codepen.io or namaa.io,facebook.org
    res.setHeader('Access-Control-Allow-Methods', 'GET, POST, PUT, DELETE, PATCH, OPTIONS');
    res.setHeader('Access-Control-Allow-Headers', 'Content-Type, Authorization');
    next();
    // use codepen.io to simulate frontend
});
